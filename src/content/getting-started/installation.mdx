		<AlertTitle>Important Note</AlertTitle>

			9ui installation relies on shadcn cli to setting up your project and

		</AlertDescription>
	</AlertContent>
</Alert>

<Steps>

<Step>
	Follow the [shadcn cli installation
	guide](https://ui.shadcn.com/docs/installation) to setup your project.
</Step>

<Step>
Install Base UI.

```bash
npm install @base-ui-components/react
```

</Step>

<Step>
Add `Root` class to your app layout.

```tsx title="layout.tsx"
<html>
	<body className="Root">{children}</body>
</html>
```

</Step>

<Step>
Update `globals.css` file.

```css title="globals.css"
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
	.Root {
		@apply bg-background isolate;
	}
}
```

</Step>

<Step>
Add 9ui theme to your project.

```bash
npx shadcn@latest add "https://9ui.dev/r/theme.json"
```

</Step>

<Step>That's it! You can now install the components.</Step>

</Steps>
